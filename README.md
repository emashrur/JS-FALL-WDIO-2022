# JS-FALL-WDIO-2022

This project is to learn WebDriver to automate web applications

Selenium -> helps automate testcases for a web-browser 
Appium -> helps automate testcases for mobile-applications

WebDriver-IO -> helps automate testcases for a web-browser and for mobile-applications

Wdio works faster compared to selenium.
Wdio is becoming famous and companies prefer to use it over selenium due to faster executions.
